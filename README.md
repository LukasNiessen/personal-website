# Personal Website

My personal homepage and blog: https://lukasniessen.com

Built with Astro, Tailwind CSS, TypeScript, and React. Fully static, hosted on AWS (S3 + CloudFront), deployed via push-based GitOps. Infrastructure lives in a [separate repo](https://github.com/LukasNiessen/lukasniessen.com-infrastructure).

Feel free to use this repo however you like, privately or commercially.

## Dev Commands

| Command            | What It Does                               |
| ------------------ | ------------------------------------------- |
| `npm i`            | Install dependencies                        |
| `npm run dev`      | Start local dev server at `localhost:4321`  |
| `npm run build`    | Build production site into `./dist/`        |
| `npm run preview`  | Preview the production build locally        |
